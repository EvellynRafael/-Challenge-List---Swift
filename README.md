# -Challenge-List---Swift
A list of programming challenges and activities using Swift

## Challenge 1 - Hot Screen
Given a variable that stores a temperature value in Celcius, create another variable that converts that temperature from Celsius to Fahrenheit.

## Challenge 2 - Calculating the Avarage
Create three variables representing the grades of three different students. Next, create a variable that calculates the avarage of these grades and displays that avarage in the console
